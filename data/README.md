# Dados locais

`db.json` é um arquivo local simples que guarda uma estrutura antiga de clientes.

Ele não é o banco principal da aplicação atual. Clientes e agendamentos ativos são armazenados no Supabase. O arquivo permanece apenas como referência e compatibilidade com versões anteriores.

Não use este arquivo para guardar senhas, documentos ou informações clínicas.
