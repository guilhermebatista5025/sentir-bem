# JavaScript do site

JavaScript controla as partes interativas que rodam no navegador do visitante.

- `script.js`: menu, formulários, notificações, carrossel, perguntas frequentes e chatbot visual.
- `aos.js`: biblioteca externa que cria animações quando elementos entram na tela.

O `script.js` pertence ao projeto e pode ser alterado. O `aos.js` é código de terceiros e deve ser mantido como está sempre que possível.
