# Identidade visual

Esta pasta guarda os logotipos da marca Sentir Bem.

- `logo-oficial.png`: usado no menu, chatbot e rodapé.
- `logo-transparente.png`: versão usada na página de login.

Esses arquivos são compartilhados por várias partes do site, por isso ficam numa pasta de identidade em vez de pertencerem a uma única seção.
