# Imagens do chatbot

Arquivos usados ou preparados para a apresentação visual do chatbot e dos planos relacionados a ele.

- `elementos-celular.png`: composição visual para a área do celular.
- `wireframe.png`: rascunho da estrutura da interface.
- `mensalidade-1].png` e `mensalidade-2.png`: artes relacionadas a opções de mensalidade.
- `ChatGPT Image ...png`: arte conceitual gerada durante o desenvolvimento.

Nem toda imagem precisa estar ativa no site; algumas servem como referência para futuras versões.
