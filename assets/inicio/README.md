# Imagens da seção inicial

Esta pasta reúne imagens preparadas para a seção `inicio`, também chamada de hero ou capa do site.

- `hero_bg.png`: imagem de fundo para telas maiores ou uso alternativo.
- `hero_mobile_bg.jpg`: fundo otimizado para telas menores, chamado pelo CSS responsivo.

Atualmente a seção também possui um vídeo externo como fundo principal.
