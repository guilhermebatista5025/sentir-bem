# Estilos do site

CSS é a linguagem que controla a aparência das páginas: cores, fontes, tamanhos, espaçamento, animações e adaptação para celular.

- `style.css`: estilos criados especificamente para o projeto Sentir Bem.
- `aos.css`: estilos da biblioteca externa de animações AOS.

O arquivo `aos.css` é de terceiros e normalmente não deve ser editado. Personalizações devem ser feitas em `style.css`.
